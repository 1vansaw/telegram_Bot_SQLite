Параметры (Sinumerik 840d sl)
9006 - Время затемнения экрана в минутах
9009 - CAPSLOCK (0 - выкл., 2 - вкл.)
9110 - уровень доступа для кнопки выход в HMI (1 - SUNRISE)
9900 - переключить на текстовое описание параметров (0 - сис. переменные, 1 - текстовые)
10000 - Названия осей
10192 - Время переключения ступени, после которого возникает ошибка.
10715[0] - М функция вызова подпрограммы (обычно указано 6).
10716[0] - Название подпрограммы для М функции из 10715 (например для М6 указано L6).
10720 - режим по умолчанию после загрузки(0-Auto, 2-MDI, 6-JOG и т.д.)
11330 - настройка инкрементов для ручного перемещения (1, 10, 100, 1000 и т.д)
14504 - Количество пользовательских данных INT (до 256)
14506 - Количество пользовательских данных HEX (до 256)
14508 - Количество пользовательских данных FLOAT (до 32)
14510 - Пользовательские данные INT (если 14504 > 0)
14512 - Пользовательские данные HEX (если 14506 > 0)
14514 - Пользовательские данные FLOAT (если 14508 > 0)
19320 и 19321 - включение/отключение технологических функций.
19730 - включение/отключение HMI функций
20000 - Название канала
20060 - Название осей в канале
20070 - Номер осей в канале
20310 - Включение/отключение функций управления инструментом
21160 - Скорость перемещения в режиме JOG с RAPID (мм/мин.)
21165 - Скорость перемещения в режиме JOG без RAPID (мм/мин.)
28020 - Количество пользовательскх данных LUD
28050 - Количество R - параметров.
30130 - Тип выходного сигнала (0 - симуляция, используется в паре с 30240)
30200 - Количество энкодеров для выбранной оси.
30240 - Тип энкодера (0 - симуляция, 1- генератор сигнала, 5 - абсолютный энкодер)
31000 - Тип энкодера для определения фактической позиции (0 - поворотный, 1 - линейка)
31010 - Расстояние между метками (только для линейки).
31020 - Количество делений датчика на оборот (только для круговых энкодеров).
31040 - Как подключен энкодер (1 - датчик подключен к машине без редуктора, 0 - датчик подключен к двигателю).
31044 - Если энкодер подключен к редуктору.
31090 - Растояние перемещения в инткрементальном режиме (1, 10, 100 и т.д.)
32000 - Максимальная скорость для перемещения оси
32010 - Скорость перемещения оси в JOG с RAPID и подачей в 100%.
32020 - Скорость перемещения оси в JOG с подачей в 100%.
32070 - Скорость перемещения оси с маховика.
32110 - Направление отсчета сигналов с датчика (-1 - против движения,  0 или 1 по движению)
32200 - Коэффициент усиления сервопривода (для К = 1, V = 1м/мин.)(Работает только если включен 10220 бит 9)
32260 - Скорость вращения двигателя.
32300 - Максимальное ускорение для выбранной уси.
32450 - Компенсация люфта (+ если датчик впереди детали станка, иначе -)
33050 - Расстояние, которое должна пройти ось, чтобы включился импульс смазки.
34000 - Ось с кулачком (1 - есть кулачок с опорной точкой, 0 - нет кулачка)
34010 - Указание направления приближения к кулачку (0 - плюсовое, 1 - минусовое направление)
34020 - Скорость перемещения к базовой точке
34030 - Максимальное расстояние до опорного кулачка (не для линеек с дист.кодированным расстоянием)
34040 - Скорость подхода к базовой точке.
34060 - Максимальное расстояние от кулачка до метки датчика после наезда на кулачок (если нет то ошибка 20002 или 20004).
30070 - Скорость перемещения от кулачка до метки датчика.
34080 - Расстояние опорных точек
34090 - Абсолютное смещение
34100 - Координата, которая устанавливается после reference.
34200 - Режим реферирования (0 - абсолютный, 1 - инкрементальный, 2 - линейка с ДКМ)
34210 - Состояние абсолютного энкодера (0 - не юстирован, 1-юстировка включена но не закончена, 2-юстирован)
34220 - Число импульсов абсолютного  датчика
34300 - Расстояние между контрольными метками
34310 - Расстояние между метками
35010 - Параметры переключения ступеней передачи (бит0=0 и бит1=0-одно передаточное число,  бит0=1 - доступны М40, М41 и т.д)
35035 - Включение/отключение функций шпинделя.
35090 - Число ступеней передач
35100 - Максимальная скорость шпинделя
35110 - Максимальная скорость шпинделя для для переключения ступеней
35120 - Минимальная скорость шпинделя для переключения ступеней
35130 - Максимальная скорость для каждой из ступеней.
35150 - Допуск на частоту вращения шпинделя
35400 - Скорость колебаний шпинделя при переключении передач.
35440 и 35450 - Время колебаний шпинделя при переключении
36030 - Допуск смещения оси при отсутствии движения
36100 и 36110 - Программный концевые выключатели
36902 - Какая ось (0 - линейная, 1 - круговая. Должно совпадать с параметром 30300)
36998 - Актуальные чексуммы (бит0 - осевые положения, бит1 - аппаратная конфигурация, бит2 - привода)
36999 - Сохраненные чексуммы.
43420 и 43430 - Ограничение рабочей зоны
51044 - 51225 - Настройка уровней доступа к различным фунцкциям
51220 - Уровень доступа с которого видна функция Relocate в стойке (0 - siemens, 1 - Manufact. и т.д.)
52210 - Скрытие функций со стартового экрана (TSM и.т.д)
52250[0] - М код для открытия шпинделя без вращения
52252[0] - М код для закрытия шпинделя без вращения
52270 и 54215 - Скрытие функций управления инструментом
52270 (бит 8) - скрыть функцию Relocate из интерфейса.
